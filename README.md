## v126 수정
- 회원 삭제 시 주문서 작성에 남아있던 회원 정보/카트/수정모드 초기화
- 주문서 작성 저장회원 드롭다운이 회원 불러온 뒤에도 계속 보이도록 수정
- 회원 검색어를 이름+전화뒷자리처럼 입력해도 검색되도록 수정
- 저장회원 중복 표시 방지(동일 이름+전화뒷4자리 기준 최신 회원만 표시)
- 회원정보 탭에 회원 목록 표 추가, 클릭 시 해당 회원 주문내역 표시
- 회원정보 탭에서 회원 삭제 버튼 표시
- 주문은 삭제하지 않고 회원 정보만 삭제
- 빌드 성공 확인
