# 여깁니다유 운영 프로그램 v140

## 변경
- 재고관리에서 상품 행 클릭 시 목록 스크롤이 맨 위로 튀는 문제 보정
- 라방 상품 추가/배정 시 본재고는 차감하지 않음
- 라방 주문 상태를 `입금확인`으로 저장할 때 해당 주문 수량만 본재고 차감
- 입금확인된 주문을 취소/삭제하면 해당 주문 수량만 본재고 복구
- 라방 종료/미판매 정리는 본재고를 더하지 않고 라방 남은수량만 0 처리

## 실행
npm install
npm run dev

GitHub에 업로드 후 Vercel 연결
