# 여깁니다유 운영 프로그램 v164

## 실행
npm install
npm run dev

## v164 수정
- 라방 주문관리 검색에서 상품명으로 주문 찾기 가능
- 주문관리 구매자명 아래에 수정 버튼 배치
- 주문관리/회원정보 상품 칸은 긴 목록 대신 [상품보기] 버튼으로 팝업 표시
- 주문 상태에 출고준비 추가
- 출고준비/입금후합배송 저장 시 택배접수 목록으로 자동 이동
- 택배접수에서 라방 주문 출고확정 시 주문 상태가 출고완료로 자동 변경
- 회원 저장/주문 저장 시 우편번호/기본주소/상세주소 유지
- 같은 회원의 킵/미출고 주문을 주문서 작성 영역에서 더 잘 보이게 표시
