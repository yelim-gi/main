# 여깁니다유 운영 프로그램 v165

## v165 수정
- 라방 주문관리에서 상품명 검색 시 해당 상품이 들어간 주문만 검색되도록 유지
- 주문관리 상품 칸은 긴 상품목록 대신 [상품보기] 버튼으로 표시
- 주문자명 아래에 [수정] 버튼 배치 유지
- 회원저장 시 우편번호/기본주소/상세주소 함께 저장
- 회원정보/주문서 작성에서 킵중 기간 표시
- 입금후합배송 주문과 기존 킵 주문을 묶어 택배접수로 보낼 수 있는 합배송 진행 기능 추가
- 택배접수에서 합배송 출고확정 시 묶인 주문들이 모두 출고완료로 변경
